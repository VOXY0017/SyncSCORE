# **App Name**: ScoreSync

## Core Features:

- Player Registration: Allow users to register new players, storing data in Firestore.
- Real-Time Ranking Display: Fetch player data from Firestore and display a real-time, automatically updating ranking list.
- Point Management: Implement +Win and -Lose buttons for each player, updating the player's score in Firestore.
- Add/Delete Player: The system allows adding and removing a player.
- Real-Time Ranking Updates: Automatically update the ranking list via Firestore listeners without page reloads.
- Input Validation: Validates new names.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and reliability in the scorekeeping.
- Background color: Light gray (#F5F5F5), a muted tone, allowing the blue and data to stand out without distraction.
- Accent color: Complementary orange (#FF9800) is used for interactive elements.
- Body and headline font: 'Inter', a sans-serif font that has a modern and neutral appearance, making the data easily readable.
- Clean, tabular layout for ranking display.
- Subtle animations when scores update to provide a visual cue to changes.
